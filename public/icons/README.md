# App Icons

These are **placeholder icons** for development.

Before production release, replace with properly designed icons:

- `icon-192.png` — 192×192 standard icon
- `icon-512.png` — 512×512 high-res icon
- `icon-maskable-192.png` — 192×192 with safe zone for adaptive icons
- `icon-maskable-512.png` — 512×512 with safe zone for adaptive icons

The SVG files can be kept as favicon sources for modern browsers.
